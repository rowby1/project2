## 개발환경
* Java JDK 1.8.0_301
* Eclipse IDE for Enterprise Java and Web Developers version 2021-06-R
* Oracle Database 11g Express Edition 11.2.0.2
* Tomcat 9.0.54

## 사용 기술
* JSP
* AJAX
* Servlet
* JavaScript
* jQuery
* Chart.js
* Bootstrap5
* HTML Geolocation API
* kakao map API
* 게시판 댓글처리
* 게시판 조회수를 활용한 인기글 
* AJAX를 활용하여 실시간 채팅 구현
